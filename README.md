# React Quiz App 💎

A Simple React Quiz App which contains MCQ’s which have 4 options in each.

### Version

1.0.0

## 📝 Usage

### Clone this Repo

```
git clone https://github.com/staticGuru/React_quiz_app.git
```

### Installation

```sh
$ npm install
```

### ✔️ Run

Then go to http://localhost:8000

```sh
$ npm start
```
